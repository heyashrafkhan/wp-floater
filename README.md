# WP Floater

WP Floater is a lightweight WordPress plugin that adds floating contact buttons (Phone and WhatsApp) to the front-end of your website, making it easy for visitors to get in touch with you.

## Features

- **Floating Contact Buttons**: Adds professional-looking floating buttons for Phone and WhatsApp.
- **Easy Configuration**: Simple settings page in the WordPress dashboard to manage your contact numbers.
- **Lightweight**: Minimal impact on page load speed with inline CSS.
- **International Format Support**: Specifically designed to handle international phone numbers for WhatsApp.

## Installation

1. Upload the `wp-floater` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **WP Floater** in the admin menu to configure your phone and WhatsApp numbers.

## Configuration

- **Phone Number**: Enter the phone number you want to be dialed when the phone button is clicked.
- **WhatsApp Number**: Enter your WhatsApp number in full international format (e.g., `971XXXXXXXXX`). Omit any leading zeros, brackets, or dashes.
